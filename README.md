# MarketPulse

A modern web application with React frontend and backend services.

## Project Structure

```
marketPulse/
├── frontend/          # React + TailwindCSS frontend application
├── README.md
└── .gitignore
```

## Frontend Setup

The frontend is built with React and styled with TailwindCSS.

### Getting Started

1. Navigate to the frontend folder:

```bash
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Tech Stack

- **Frontend**: React, TailwindCSS, Vite
